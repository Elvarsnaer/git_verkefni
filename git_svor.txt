Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd = breytir um directory
	cd ~ = breytir yfir í home directory
	ls = birtir lista um efni í file-um og directories
	pwd = birtir fulla nafnið á working-directory
	mkdir = býr til directories og files

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone = klónar repository í nýtt directory
	git log = sýnir commit log
	git status = sýnir working tree status
	git diff = sýnir breytingar á milli commits, commit og working tree, o.s.f.v.
	git checkout = skiptir um branch eða restorar working tree files

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff = sýnir breytingar á milli commits, commit og working tree, o.s.f.v.
	b) git diff --staged = sýnir hvað hefur verið stage-að fyrir næsta commit
	c) git diff commit1 commit2 = sýnir muninn á tveimur commits

4. 	Hvað er útgáfustýring (e.version control)? = a system that records changes to a file or set of files over time so that you can recall specific versions later.
